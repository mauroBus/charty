ChartInterface Class

Defined in: src\api\chartinterface.js:1

Sets an interface for adding a link between the chart and the data accessor.

Uses an event manager for defining different charty events. Since events need to be present when chart is rendered, for attachment to every SVG node, they should be defined by draw method. This makes an easy way of propagating events to each base rendering class.

Note : events are NOT defined in chart init, it can happen that, at this point, events handler are not yet defined or they don't have all necessary data.
