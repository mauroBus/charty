Axis Class

Defined in: src/components/axis/axis.js:1

Basic Axis representation.

Only one X/Y is sufficient for chart drawing, but can contain more. The idea is to draw an axis and locate it wherever is necessary.

Wether Axis is a BaseChart, no need to extend it, since it will implement all the functions needed.
