DataMapper Class

Defined in: src\composition\datamapper\datamapper.js:1

Maps every data point, taking into account one to many scales. Works as a scale composite object.

It is possible to requiere more than one scale to map certain elements, so this component will make the correct positioning.

Each chart will have two mappers : one for horizontal axis and other for vertical axis.
